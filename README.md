# FortiGate FortiClient VPN Setup Guide

This repository contains documentation for configuring SSL-VPN access using FortiGate and FortiClient.

## Requirements

- FortiGate firewall
- FortiClient VPN
- Administrative access to FortiGate
- Public IP address or FQDN
- Internet connection

## Topics Covered

- Creating SSL-VPN users
- Installing FortiClient
- Connecting remote users
- LDAP integration
