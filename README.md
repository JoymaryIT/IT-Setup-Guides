# IT-Setup-Guides

This repository contains documentation for configuring:
- SSL-VPN access using FortiGate and FortiClient
- VoIP Setup using Unifi

## Requirements

**SSL-VPN access using FortiGate and FortiClient**
- FortiGate firewall
- FortiClient VPN
- Administrative access to FortiGate
- Public IP address or FQDN
- Internet connection
- FortiClient Download Link: https://www.fortinet.com/support/product-downloads
